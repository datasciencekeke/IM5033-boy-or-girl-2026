**File Source**
Download the datasets from the in-class Kaggle competition and put them in this folder.
Codes use this directory for datasets.

**Data fields**
The fields are fairly self-explanatory. The training set and the test set contain the same fields with the target field (gender) in the test set being zero-valued.

**Disclaimer**
The data was collected with the consent of the respondents and the dataset does not contain any PII or Personally Identifiable Information.
